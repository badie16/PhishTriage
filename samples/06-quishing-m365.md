# Sample 06 — Quishing : QR Code Microsoft 365

## Raw Message

```
From:     "Microsoft 365 Security" <noreply@m365-alerts.tk>
Reply-To: security@m365-alerts.tk
To:       utilisateur@entreprise.fr
Subject:  Authentification requise — Mise à jour de sécurité

--- Message Body ---

Bonjour,

Votre session Microsoft 365 nécessite une vérification de sécurité.
Pour des raisons de sécurité renforcée, les liens directs sont
désactivés. Scannez le QR code ci-dessous avec votre téléphone
pour confirmer votre identité :

[ IMAGE : QR CODE ]

📱 Scannez avec votre appareil mobile pour accéder à la page
de connexion sécurisée Microsoft.

⏰ Cette demande expire dans 2 heures.

Merci,
L'équipe Microsoft 365 Security

--- Note ---
Le QR code pointe vers : https://microsoft-365-verify.tk/login
(domaine .tk ≠ microsoft.com)
```
