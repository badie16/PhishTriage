# 📊 Phishing Analyses

Complete analysis reports for each sample. Every report follows the same structure:

1. **Sample reference**
2. **Header analysis** (From, Reply-To, Return-Path, SPF/DKIM/DMARC)
3. **Red flags** identified
4. **Cognitive triggers** exploited
5. **Threat classification** (Safe / Suspicious / Malicious)
6. **Triage decision** (Close / Warn User / Block & Escalate)
7. **IOCs extracted**
